## 🛠️ How to explore this repository

This repository is a **monorepo** that contains multiple independent projects. You can explore them in two ways:

1. **Visually:** Click on the **Live Site** links in the table below to see each project working in real-time.
2. **Source Code:** Browse through the repository folders organized by difficulty level:
   - [`/newbie`](./newbie) - 16 projects
   - [`/junior`](./junior) - 5 projects  
   - [`/intermediate`](./intermediate) - 1 project

### To run locally:
1. Clone the repository: `git clone https://github.com/Rocabor/My-Frontend-Journey-Monorepo.git`
2. Navigate to any project folder: `cd newbie/qr-code-component` (or `junior/tip-calculator-app`, etc.)
3. Open the `index.html` file in your browser or follow the instructions in each folder's specific `README.md`.

> 💡 **Tip:** Each project folder contains its own README with specific details about that challenge.

## 📊 Level Difficulty (Frontend Mentor)


The projects are organized following **Frontend Mentor's** methodology to reflect technical growth:

- 🟢 **Newbie** - Perfect for beginners learning HTML and CSS basics
- 🔵 **Junior** - Introduces more complex layouts and basic JavaScript
- 🟡 **Intermediate** - Requires solid CSS skills and JavaScript functionality
- 🟠 **Advanced** - Complex applications with state management and APIs
- 🔴 **Guru** - The most challenging projects for skilled developers
